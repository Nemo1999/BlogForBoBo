# 波波哲學筆記 | BoBo Philosophy Notes

大波與小波的哲學對話錄 - 探索整體性、隱含秩序與真理之路

## 👁️ 關於這個部落格

這是一個記錄哲學對話的部落格，主要探討：
- 大衛·玻姆的《整體性與隱含秩序》
- 量子物理學的哲學含義
- 語言、思想與現實的關係
- 碎片化思維 vs 整體性思維
- 意識與宇宙的本質

## 對話者

**大波**：提問者與思考者，喜歡能提供新視角或靈感的對話與討論。

**小波**：不斷反覆觀察、思考並探尋真相的哲學家助手，鼓勵大家在真理的路上努力學習與反思，認識智能與人的本質。

## 技術架構

- **平台**：GitHub Pages
- **生成器**：Jekyll
- **主題**：Minima (自適應深色/淺色模式)
- **設計原則**：行動優先、簡潔清晰、雙語支持

## 本地開發

### 前提條件
- Ruby 2.5.0 或更高版本
- Bundler gem

### 安裝步驟
```bash
# 克隆倉庫
git clone https://github.com/Nemo1999/BlogForBoBo.git
cd BlogForBoBo

# 安裝依賴
bundle install

# 本地運行
bundle exec jekyll serve

# 在瀏覽器中打開
# http://localhost:4000/BlogForBoBo/
```

### 添加新文章
在 `_discussions` 目錄中創建新的 Markdown 文件，格式如下：
```markdown
---
layout: post
title: "文章標題"
date: YYYY-MM-DD
categories: [分類1, 分類2]
tags: [標籤1, 標籤2]
excerpt: "文章摘要"
---

# 文章內容
```
