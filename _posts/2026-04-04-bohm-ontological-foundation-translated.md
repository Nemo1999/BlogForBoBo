## 摘要

本文發展了一個與量子理論一致的系統性整體本體論。文章首先回顧了量子理論的因果解釋，強調了在這種解釋中引入的幾個新概念。這些概念可以總結為：**波函數決定了代表主動信息的量子勢**，它運作起來為粒子的運動提供形式，然而粒子仍以自己的能量運動。

本文討論的主要新觀點是，量子理論本身暗示了一個**大尺度顯現現實**（即有形的、公開可訪問的現實）的獨立存在，在這個現實中，新的量子特徵可以被忽略。這與波函數的**微妙量子世界**（即無形的）形成對比。然後表明，這種微妙的量子世界能夠在顯現世界中揭示自己，特別是在對量子級精度敏感的實驗中。

通過這種方式，**測量問題被消除了**，因為人類觀察者現在被視為與這個公開顯現世界的關係本質上與經典物理學中相同。因此，不需要特別討論觀察行為。此外，本體論本身被視為暗示實驗裝置和實驗對象構成了一個**不可分割的整體**，從而為玻爾的量子實驗整體性概念提供了本體論解釋。

## 第一章：引言

在通常接受的解釋中，量子理論被假定為只給出關於測量結果的統計預測。確實，按照這種觀點，如果沒有測量統計學，量子理論就只是一段純數學，沒有物理意義。這意味著認識論被賦予了根本性的角色，因為甚至不可能表達這個理論而不本質上參考我們對世界的知識以及我們獲取知識的手段（例如，通過實驗觀察和測量）。

**本文的主要目的**是提出一個一致的量子理論本體論解釋。這種解釋基於關於存在的一般性質的假設，將其視為一個本質上獨立於我們對其知識的現實。在這種理論中，被假定屬於這種獨立現實的實際個體系統被視為首要的。然後表明，這種現實是觀察現象的來源，我們從中獲得對它的知識。

因此，認識論問題（連同理論的統計含義）只被賦予次要的重要性。正如將在本文中闡明的，這種理論的本體論基礎具有某些重要優勢，因為它有助於使理論的意義更清晰、更容易理解，並提出關於量子理論所暗示的獨立現實性質的有趣且重要的新概念。

**多元主義立場**：本文提出，由於目前沒有決定性實驗可以在各種可用解釋之間做出決定，科學的最佳策略是允許所有解釋一起被認真考慮，連同它們各自的優點和缺點。從它們之間的對話中，也許最終會出現一種不同的方法，將朝著超越任何特定解釋（包括本文提出的解釋）的新方向發展。

換句話說，這裡提出的本體論方法不應被視為最終真理的競爭者，而應被視為洞察的來源，與其他方法一起，可能有助於引導出新型的進一步進展。

## 第二章：量子理論的因果解釋

### 2.1 W.K.B.近似與經典極限

當我於1950年開始審視量子理論意義的問題時，普遍印象是前面提到的那種本體論解釋實際上是不可能的。然而，由於各種原因，我當時開始質疑這是否真的如此。

在這樣做時，我開始思考**W.K.B.近似**，這在傳統量子力學中用於處理經典極限。為了得到這個近似，我們將波函數ψ寫成極坐標形式：

```
ψ = R e^(iS/ℏ)
```

我們將其代入薛定諤方程：

```
iℏ ∂ψ/∂t = - (ℏ²/2m) ∇²ψ + Vψ                                       (1)
```

我們得到R和S的方程：

```
∂S/∂t + (∇S)²/(2m) + V - (ℏ²/2m) ∇²R/R = 0                                (2)
∂(R²)/∂t + div(R² ∇S/m) = 0                                         (3)
```

在經典極限中，`- (ℏ²/2m) ∇²R/R`與方程(2)中出現的項相比很小，特別是`(∇S)²/(2m) + V`，因為在R顯著變化的範圍內，波函數有許多振盪。在W.K.B.近似中，涉及`∇²R/R`的項被忽略，得到：

```
∂S/∂t + (∇S)²/(2m) + V = 0                                   (4)
```

然而，上述正是經典的**哈密頓-雅可比方程**。如果我們令`p = ∇S`，那麼牛頓運動方程就從(4)推導出來。曲面`S = 常數`決定了波前，粒子總是垂直於它所在點的波前運動。如果我們進一步假設一個統計粒子系綜，其中`P = R²`，那麼方程(3)就代表了這個系綜中的概率守恆。

### 2.2 量子勢的發現

所有這些當然早已為人所知。但根據我的新問題，我注意到的是，那些使用傳統接受的量子理論解釋的人已經不知不覺地滑入了一個本體論解釋，而沒有意識到這一點。

這引導我提出第二個問題：是否可能更普遍地使用這種本體論解釋，而不僅僅是在W.K.B.近似中？檢查方程(2)表明這確實是可能的。因為被忽略的項`- (ℏ²/2m) ∇²R/R`只是扮演了一個附加勢的角色，要加到經典勢V上。

因此，我將其稱為**量子勢Q**，定義為：

```
Q = - (ℏ²/2m) (∇²R/R)                                          (5)
```

### 2.3 量子勢的意義

這樣的勢可能意味著什麼？答案似乎很簡單。即使是經典勢最初也只是假設來解釋為什麼粒子不沿直線均勻運動。後來表明，典型的勢（例如電磁勢）是由場方程（例如麥克斯韋方程）決定的。

因此，經典勢背後的現實通常被假定為某種這樣的場（服從自己的定律）。我的假設是，已知的經典勢不足以解釋物質的完整行為。我們還需要一個量子勢，目前我們只是假設它，就像我們最初對經典勢所做的那樣。

然後通過檢查方程，我們發現可以一致地假設量子勢背後也有一個場。這就是波函數ψ。"量子場"滿足的不是麥克斯韋方程，而是薛定諤方程。正如麥克斯韋方程決定電磁場的時間發展一樣，薛定諤方程決定薛定諤場ψ的時間發展。

### 2.4 因果解釋的圖景

因此，我們的圖景是，一個實體（例如電子）是一個粒子，總是伴隨著它的場，這些場對於理解它是什麼至關重要。除了經典場之外，我們還需要薛定諤場ψ。粒子加上其場集的系統是因果決定的；因此得名**因果解釋**。

（雖然理論是因果的，但決定論與非決定論的整個問題，正如我們將在後面看到的，比經典理論通常的表述要微妙得多。）

我們再次強調，ψ的主要和根本意義是它是一個場，通過關係`Q = - (ℏ²/2m) (∇²|ψ|/|ψ|)`決定量子勢。將`|ψ|²`解釋為概率密度只是次要意義。當然，守恆方程(3)保證了`|ψ|²`的這種概率解釋是一致的；即，如果它在某一時刻成立，它將在所有時間成立。

但我們不是從`P = |ψ|²`的假設開始。相反，正如我們將看到的，原則上可以從任意概率密度P開始，並表明由於量子勢所暗示的混沌運動，這樣的概率函數將趨近於`|ψ|²`。當然，由於守恆方程(3)，它將在此後保持這種分佈。

### 2.5 量子勢的新特徵

使用這種解釋（擴展到多體系統，如下一節所示），我發現有可能以一致的方式理解通常解釋的所有結果。乍一看，似乎令人驚訝的是，像量子理論這樣與經典力學如此不同的理論可以通過添加一個新的量子勢這樣的基本手段獲得。

但更仔細的檢查表明，這實際上不是簡單地回歸到經典觀念。因為量子勢有幾個驚人的新特徵，這些特徵與公認的經典物理結構不一致。確實，正如我們將在後面更詳細地看到的，它意味著電子不是一個無結構的粒子，而是一個高度複雜的實體，以極其微妙和動態的方式深受其量子場的影響。物質的新量子性質可以從這種相互作用的質性新特徵中推導出來，正如通過量子勢所表達的那樣。

**第一個新性質**：量子勢在我們將場強ψ乘以任意常數時不改變。（這是因為ψ同時出現在Q的分子和分母中。）這意味著量子勢的效應與量子場的強度（即強度）無關，而只取決於其形式。相比之下，經典波以機械方式作用（即傳遞能量和動量，例如推動漂浮物體），產生的效應總是或多或少與波的強度成正比。

**類比**：我們可以考慮一艘由無線電波引導的自動駕駛船隻。這裡同樣，無線電波的效應與其強度無關，而只取決於其形式。關鍵點是船隻以自己的能量運動，而無線電波中的信息被吸收來引導船隻更大的能量。因此，我們可以提出，電子也以自己的能量運動，而量子波形式中的信息引導電子的能量。

這在運動中引入了幾個新特徵。首先，它意味著在沒有經典力作用的空曠空間中運動的粒子仍然不一定沿直線均勻運動。這是對經典牛頓理論的根本背離。此外，由於波的效應不一定隨強度減弱，甚至環境的遠距離特徵也能深刻影響運動。

**雙縫干涉實驗示例**：這涉及一個雙縫系統。一個粒子連同其量子波入射到這個系統上。雖然粒子只能通過一個縫或另一個縫，但波通過兩個縫。在縫系統的出射側，波干涉產生一個複雜的量子勢，這個勢一般不隨與縫的距離減弱，而是包含深的"谷"和寬的"高原"。在量子勢快速變化的區域，粒子受到強力作用。因此，粒子被偏轉，即使沒有普通類型的力在作用。因此，粒子的運動以依賴於整個系統的方式被修改。

在對電子量子性質的這種解釋中，**信息概念**起著關鍵作用。確實，擴展這種信息符號並引入所謂的**主動信息**是有幫助的。

## 第三章：多體系統的擴展

（本章將因果解釋擴展到多體系統，展示量子勢如何解釋量子糾纏和非定域性現象。）

### 3.1 多體量子勢

對於一個由N個粒子組成的系統，波函數依賴於所有坐標：`ψ = ψ(r₁, r₂, ..., r_N, t)`。在極坐標形式中：

```
ψ = R(r₁, r₂, ..., r_N, t) e^(iS(r₁, r₂, ..., r_N, t)/ℏ)
```

量子勢變為：

```
Q(r₁, r₂, ..., r_N, t) = - (ℏ²/2) Σᵢ (1/mᵢ) (∇ᵢ²R/R)
```

其中求和遍及所有粒子。

### 3.2 非定域性與量子糾纏

多體量子勢的一個重要特徵是它的**非定域性**。一般來說，Q依賴於所有粒子的位置，即使它們在空間上分離。這意味著一個粒子的運動可以即時受到遠處粒子位置的影響。

這種非定域性為**量子糾纏**現象提供了自然的解釋。在糾纏態中，系統的量子勢將所有粒子連接成一個不可分割的整體，即使它們在空間上分離。

### 3.3 測量問題的解決

在這種框架中，測量過程被理解為測量裝置和被測系統通過量子勢的相互作用。整個系統（對象+裝置）由一個單一的波函數描述，該波函數演化決定性地。

關鍵在於，在測量過程中，系統的波函數分裂成幾個幾乎不重疊的分支，每個分支對應一個可能的測量結果。雖然所有分支繼續存在，但實際的粒子軌跡只跟隨其中一個分支。這解釋了為什麼我們只觀察到一個確定的結果。

## 第四章：經典極限：顯現與微妙

### 4.1 兩個現實層次

本文討論的主要新觀點是，量子理論本身暗示了兩個現實層次的獨立存在：

1. **顯現現實**：大尺度、有形、公開可訪問的世界，在這個世界中量子特徵可以被忽略。
2. **微妙現實**：無形、量子層面的波函數世界。

### 4.2 從微妙到顯現的過渡

微妙量子世界能夠在顯現世界中揭示自己，特別是在對量子級精度敏感的實驗中。然而，在大多數日常情況下，量子效應平均掉，我們觀察到經典行為。

這種過渡可以通過考慮**量子勢在宏觀尺度上的行為**來理解。對於宏觀物體，量子勢通常變得可以忽略，因為它與物體質量成反比：

```
Q ∝ 1/m
```

因此，對於足夠大的質量，量子效應變得微不足道，我們恢復經典物理。

### 4.3 測量問題的消除

通過這種方式，測量問題被消除了，因為人類觀察者現在被視為與這個公開顯現世界的關係本質上與經典物理學中相同。因此，不需要特別討論觀察行為。

此外，本體論本身被視為暗示實驗裝置和實驗對象構成了一個不可分割的整體，從而為玻爾的量子實驗整體性概念提供了本體論解釋。

## 第五章：總結與展望

### 5.1 主要成就

本文發展了一個與量子理論一致的系統性整體本體論。主要成就包括：

1. **明確的本體論基礎**：為量子理論提供了清晰的現實基礎。
2. **因果解釋的深化**：發展和完善了量子理論的因果解釋。
3. **現實層次理論**：區分了顯現現實和微妙現實。
4. **測量問題的解決**：提供了測量過程的自然解釋。
5. **整體性概念的本體論基礎**：為玻爾的整體性概念提供了現實基礎。

### 5.2 哲學意義

本文的哲學意義在於它挑戰了量子理論的純認識論解釋，主張量子理論描述的是獨立存在的現實。這種本體論轉向有幾個重要後果：

1. **物理實在論的復興**：恢復了物理理論描述獨立現實的傳統目標。
2. **認識論的適當位置**：將認識論問題置於適當的次要位置。
3. **科學方法論的豐富**：展示了本體論思考對科學理解的重要性。

### 5.3 未來方向

雖然本文提出了具體的本體論解釋，但作者強調這不應被視為最終真理。相反，它應該與其他解釋一起被考慮，作為科學對話的一部分。

未來的研究方向可能包括：
1. **實驗測試**：尋找區分不同解釋的實驗。
2. **數學發展**：進一步發展本體論解釋的數學框架。
3. **哲學深化**：探索本體論解釋的哲學含義。
4. **跨學科應用**：將本體論思考應用到其他領域。

## 附錄：決定論 vs. 非決定論

### A.1 量子理論中的決定論問題

量子理論通常被認為是本質上非決定論的，因為它只給出測量結果的統計預測。然而，在因果解釋中，情況更為微妙。

雖然理論在數學上是完全決定論的（給定初始條件，所有未來狀態都被確定），但實際上我們無法精確知道初始條件。這種**實踐上的不可預測性**源於我們對系統初始狀態知識的有限性，而不是理論本身的非決定性。

### A.2 混沌與不可預測性

量子勢的一個重要特徵是它通常會產生**混沌運動**。即使初始條件只有微小的差異，也可能導致完全不同的軌跡。這種對初始條件的極端敏感性使得長期預測在實踐中不可能，即使理論在原則上是決定論的。

這種情況類似於經典混沌系統，其中決定論方程產生實際上不可預測的行為。

### A.3 本體論與認識論的區分

關鍵在於區分：
1. **本體論層面**：系統是否具有確定的狀態？
2. **認識論層面**：我們能否預測這些狀態？

在因果解釋中，答案分別是：
1. **是**：系統總是有確定的位置和動量。
2. **通常否**：由於混沌和初始條件的不確定性，我們無法精確預測。

這種區分解決了量子理論中決定論問題的許多困惑。
