# 🔒 部落格安全設置說明

## 📋 安全系統概述

已為玻姆哲學研究部落格設置完整的訪問控制系統，以保護受版權保護的學術內容。

## 🛡️ 安全功能

### 1. 密碼保護系統
- **訪問控制**：所有頁面需要密碼才能訪問
- **密碼設置**：默認密碼為 `bohm2026`
- **Cookie 管理**：登入後24小時內有效
- **自動登出**：閒置8小時後自動登出

### 2. 訪問流程
```
用戶訪問 → 密碼保護頁面 → 輸入正確密碼 → 設置訪問Cookie → 進入受保護內容
```

### 3. 安全頁面
- **密碼頁面**：`/BlogForBoBo/password-protect.html`
- **登出頁面**：`/BlogForBoBo/logout.html`
- **受保護首頁**：`/BlogForBoBo/home-content.html`

## 🔧 技術實現

### 前端檢查
```javascript
// 訪問權限檢查
function hasAccess() {
    const cookies = document.cookie.split(';');
    for (let cookie of cookies) {
        if (cookie.trim().startsWith('bohm_access=')) {
            return cookie.trim().substring('bohm_access='.length) === 'true';
        }
    }
    return false;
}

// 沒有權限時重定向
if (!hasAccess()) {
    window.location.href = '/BlogForBoBo/password-protect.html';
}
```

### 密碼驗證
```javascript
const CORRECT_PASSWORD = 'bohm2026'; // 可以定期更換

if (password === CORRECT_PASSWORD) {
    // 設置24小時有效的Cookie
    const expires = new Date();
    expires.setTime(expires.getTime() + (24 * 60 * 60 * 1000));
    document.cookie = `bohm_access=true; expires=${expires.toUTCString()}; path=/`;
}
```

## 🔐 密碼管理

### 默認密碼
- **當前密碼**: `bohm2026`
- **有效期**: 可根據需要定期更換

### 更換密碼步驟
1. 編輯 `password-protect.html` 文件
2. 修改第 50 行的 `CORRECT_PASSWORD` 變量
3. 提交更改並部署

### 建議的密碼策略
- 每3-6個月更換一次密碼
- 使用強密碼（大小寫字母、數字、符號組合）
- 不同用戶可以分配不同密碼
- 記錄密碼變更歷史

### 定期檢查
1. **密碼有效性**：檢查密碼是否仍然安全

### 故障排除
#### 問題：無法訪問受保護內容
**解決方案**：
1. 清除瀏覽器Cookie
2. 重新訪問密碼頁面
3. 輸入正確密碼
4. 如果問題持續，檢查JavaScript是否啟用

#### 問題：密碼無效
**解決方案**：
1. 確認輸入的密碼正確
2. 檢查密碼是否已更換
3. 聯繫管理員獲取最新密碼

#### 問題：自動登出過快
**解決方案**：
1. 檢查瀏覽器設置
2. 確認沒有清除Cookie的擴展
3. 調整自動登出時間（修改protected.html中的計時器）
